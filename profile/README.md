# EnDevSols

**EnDevSols is a custom AI development company building production-ready AI agents, RAG chatbots, LLM integrations, automation systems, and scalable SaaS platforms for startups, enterprises, and growing technology teams.**

We help businesses turn AI ideas into reliable software products — from internal knowledge assistants and customer support automation to agentic workflows, document intelligence, and full-stack AI-powered platforms.

[Visit EnDevSols](https://endevsols.com/) · [Explore AI Development Services](https://endevsols.com/services) · [Contact Us](mailto:technology@endevsols.com)

## What We Build

- **AI Agent Development** — autonomous agents that use tools, APIs, memory, and business logic to complete real workflows.
- **RAG Chatbot Development** — retrieval-augmented generation systems for internal knowledge bases, document Q&A, and enterprise search.
- **Custom AI Chatbot Development** — customer support bots, lead qualification assistants, website chatbots, and multi-channel AI agents.
- **LLM Integration Services** — OpenAI, Anthropic, Gemini, AWS Bedrock, vector databases, tool calling, and AI orchestration.
- **AI SaaS Product Development** — MVPs, multi-tenant SaaS platforms, AI features, dashboards, billing, and scalable backend systems.
- **Enterprise Software Development** — custom web applications, automation platforms, APIs, cloud infrastructure, and business systems.

## Open Source by EnDevSols

We maintain open-source tools focused on production-grade AI, RAG, document intelligence, and LLM reliability.

| Project | Purpose |
|---|---|
| [LongTrainer](https://github.com/ENDEVSOLS/Long-Trainer) | Production-ready RAG framework for multi-tenant AI chatbots, streaming, vector search, tools, and persistent memory. |
| [LongSuite](https://github.com/ENDEVSOLS/Long-Suite) | Open-source ecosystem for document parsing, RAG assistants, hallucination detection, and retrieval regression testing. |
| [LongParser](https://github.com/ENDEVSOLS/LongParser) | Document intelligence engine for converting PDFs, DOCX, PPTX, XLSX, and CSV files into AI-ready chunks for RAG pipelines. |
| [LongTracer](https://github.com/ENDEVSOLS/LongTracer) | LLM hallucination detection and claim verification for RAG pipelines and AI assistants. |
| [LongProbe](https://github.com/ENDEVSOLS/LongProbe) | RAG regression testing for detecting retrieval quality issues, lost chunks, and broken knowledge pipelines. |
